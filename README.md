# KE4H - Project 1: Enriching the ArCo Knowledge Graph

## Overview

This repository contains the first project developed for the **Knowledge Engineering for Humanities (KE4H)** course.

The objective of the project is to explore the **ArCo Knowledge Graph**, identify semantic gaps through SPARQL queries, and propose RDF enrichments supported by Large Language Models (LLMs).

Our case study focuses on the **Rocca Sforzesca of Imola**, analysing how the monument is represented within ArCo and identifying missing or incomplete information.

---

## Project Goals

* Explore the ArCo Knowledge Graph through SPARQL.
* Identify semantic gaps in the representation of the Rocca Sforzesca of Imola.
* Generate RDF triples to enrich the knowledge graph.
* Evaluate the use of Large Language Models for RDF generation.
* Compare different prompting techniques and LLM behaviours.

---

## Identified Knowledge Gaps

Two main semantic gaps were identified:

### 🎬 Missing current use

Although the Rocca is officially used every summer as an **outdoor cinema (Arena Cinema)**, this current function is not represented within ArCo.

### 👑 Historical representation

The knowledge graph currently represents **Danesio Maineri** as the author of the Rocca, while **Caterina Sforza**, one of the most important historical figures associated with the fortress, is completely absent from the semantic description.
